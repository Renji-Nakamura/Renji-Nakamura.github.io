課題：
標準入力から読み込んだ整数が「正の数」「負の数」「ゼロ」のいずれかを判定して表示するプログラムを作成せよ。

要件:
scanfで 1つの整数を読み込む。
関係演算子（>, <, ==）を用いて判定する。
if ～ else if ～ else 文を使う。

/*
 * 課題番号：aa05-01
 * 概要：任意の整数を入力し、正負、ゼロを判別し、それぞれ対応するテキストを出力する。
 * 作成年月日：25/05/15
 * 学籍番号：sCAuPW3kzW
 * 氏名：Renji
 */
 
#include <stdio.h>
 
int main(void) {
    int x;
 
    printf ("整数を入力してください!"); scanf("%d", &x);
 
    if (x > 0)
        printf("正の値です！");
    else if (x < 0)
        printf("負の値です！");
    else
        printf("ゼロです!");
    return 0;
}
