scanf 関数によりキーボードから2つの整数を入力し，
前者の値が後者の何%であるかを実数で表示するプログラムを作成しなさい．
（テキスト P.37 演習2-5）
ただし，出力する実数は，小数点以下2桁までを表示すること．

/*
 * 課題番号：a04-02
 * 概要：実数を入力し、百分率を出力!
 * 作成年月日：25/05/08
 * 学籍番号：z7yjf2eREe
 * 氏名：Renji
 */
 
#include <stdio.h>
 
int main(void)
{
    int x,y;
    printf("二つの実数x,yを入力してね!\n");
    printf("実数x");scanf("%d",&x);
    printf("実数y");scanf("%d",&y);
    double a,b;
    a=x;
    b=y;
    printf("前者は後者の何パーセントかを出力しないといけない課題なので.....");
    printf("前者は後者の""%.2f%%です",a/b*100);
    return 0;
}
